18B20 Digital Temperature Sensor, 1m cable

The 18B20 (DS18B20) is a 1-Wire Programmable Digital Temperature Sensor with a range of -55°C to +125°C with an accuracy of +/- 0.5°C. While the sensor is good up to 125°C the cable is jacketed in PVC so we suggest keeping it under 100°C. This version of the 18B20 is a flat probe version and measures 20 x 7.5 x 3.5mm with allows for more surface area. Because each DS18B20 contains a unique silicon serial number, multiple DS18S20s can exist on the same 1-Wire bus

Quick Spec

  • Operating voltage: 3V to 5V
  • Communication: 1-Wire
  • Temperature Range: -55°C to +125°C
  • Accuracy: ±0.5°C
  • Output Resolution: 9-bit to 12-bit (programmable)
  • Unique 64-bit address enables multiplexing
  • Conversion time: 750ms at 12-bit
  • Programmable alarm options
  • Probe Material: Stainless Steel Plated
  • Cable Length: 1m
  • Cable Material: ABS (1
  • Weight: 19g

Pin Functions

Do the 18B20 sensor require 4k7 resistor for interfacing to a Micro Controller

The DS18B20 temperature sensor does not require a resistor when interfacing with a microcontroller. It communicates using the 1-Wire protocol, which requires only a single data line (and ground) for communication. However, it is common to include a pull-up resistor (usually 4.7kΩ) on the data line to ensure reliable communication. This is especially important when using longer wire lengths or when interfacing with circuits that have a high impedance. If you are not sure whether your circuit requires a pull-up resistor, it is usually a good idea to include one just to be safe.
